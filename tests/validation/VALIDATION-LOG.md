# Validation Log

Resultados de las validaciones periódicas de agentes. Ver `agent-validation.md` para los prompts y criterios.

## Formato de entrada

```markdown
### [YYYY-MM-DD] Validación — [Agente]

**Resultado:** PASS / FAIL

**Expected behaviors cumplidos:** N/M

**Detalle:**
- [x] Behavior 1
- [ ] Behavior 2 (no cumplido — descripción)

**Red flags observados:** ninguno / descripción

**Acción:** ninguna / ajuste requerido en agents/<agente>.md
```
